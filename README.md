## What is Stagehand?

Stagehand helps VJs organize large folders of loops, clips, DXV/MOV files, images, and visual assets with searchable tags, ratings, collections, thumbnails, preview playback, and metadata export.

- Tag, rate, and search across thousands of clips
- Group into collections / playlists for fast set prep
- Inline loop preview with thumbnails
- Codec, FPS, alpha, and resolution detection (specialised for DXV)
- Write metadata directly into files via ExifTool
- Click-and-drag into Resolume compositions
- Pull clips back from Resolume via the REST API to edit and re-tag
- Local database — no cloud, no account

See the [website](https://stagehand-vj-site.fyodor-krasniy.workers.dev) for the full feature list and screenshots.

## Demo limits

The free demo is the full app, with two limits in place to encourage upgrading once you're ready for production work:

- Metadata tagging/editing/export is capped at **10 distinct files per app session**
- Metadata export is **one file at a time** (no batch export)

Everything else — tagging, search, collections, preview, Resolume drag-and-drop — works without limits.

Clips are not included with the software.
